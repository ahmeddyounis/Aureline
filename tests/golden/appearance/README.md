# Appearance goldens

This directory holds **appearance** golden baselines for protected shell
surfaces (screenshot captures + token-adoption baselines).

Goals:

- make theme/density/reduced-motion drift measurable early;
- keep the harness narrow (protected M1 shell surfaces only);
- keep the truth source shared (semantic tokens + appearance session records).

Subtrees:

- `shell_chrome/` — initial protected-row surface capture + token-adoption audit
  seed (dark/light/high-contrast + a density or reduced-motion variant).
