# Onboarding Help-Pack Beta Fixtures

Reviewable fixtures for the beta onboarding/help-pack manifest implemented in
[`crates/aureline-shell/src/help_packs/onboarding_beta.rs`](../../../../crates/aureline-shell/src/help_packs/onboarding_beta.rs).

The JSON files are generated by the headless inspector
[`crates/aureline-shell/src/bin/aureline_shell_onboarding_help_pack_beta.rs`](../../../../crates/aureline-shell/src/bin/aureline_shell_onboarding_help_pack_beta.rs).

## Index

| Fixture | Coverage |
| --- | --- |
| [`manifest.json`](./manifest.json) | Versioned pack and item IDs, glossary-pack IDs, citations, release-truth badges, locale fallback, mirror/offline posture, command hints, and state refs. |
| [`surface_projection.json`](./surface_projection.json) | Start Center, migration center, Help search, contextual why-now, and recovery-first surface coverage. |
| [`support_export.json`](./support_export.json) | Metadata-only support export that records active pack versions, fallback state, dismissed/helpful state refs, and omits raw bodies. |
| [`source_language_fallback.yaml`](./source_language_fallback.yaml) | Locale fallback proof for the command palette migration bridge. |
| [`offline_mirror_fallback.yaml`](./offline_mirror_fallback.yaml) | Mirror/offline fallback proof for docs browser help. |

## Regenerate

```sh
cargo run -q -p aureline-shell --bin aureline_shell_onboarding_help_pack_beta -- manifest \
  > fixtures/help/m3/onboarding_help_packs/manifest.json
cargo run -q -p aureline-shell --bin aureline_shell_onboarding_help_pack_beta -- surfaces \
  > fixtures/help/m3/onboarding_help_packs/surface_projection.json
cargo run -q -p aureline-shell --bin aureline_shell_onboarding_help_pack_beta -- support-export \
  > fixtures/help/m3/onboarding_help_packs/support_export.json
```

## Verification

```sh
cargo test -p aureline-shell --test onboarding_help_pack_beta_fixtures
cargo run -q -p aureline-shell --bin aureline_shell_onboarding_help_pack_beta -- validate
```
