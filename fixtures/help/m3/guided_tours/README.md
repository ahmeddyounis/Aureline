# Guided Tours And Learning Mode Fixtures

Reviewable fixtures for the beta guided-tour, guided-exercise rail, learning-mode profile, progress snapshot, and support-export records implemented in [`crates/aureline-shell/src/learning_mode/mod.rs`](../../../../crates/aureline-shell/src/learning_mode/mod.rs).

The JSON files are generated by the headless inspector [`crates/aureline-shell/src/bin/aureline_shell_learning_mode_beta.rs`](../../../../crates/aureline-shell/src/bin/aureline_shell_learning_mode_beta.rs).

## Index

| Fixture | Coverage |
| --- | --- |
| [`manifest.json`](./manifest.json) | Versioned tour packages, stable targets, citations, exercise rails, profile controls, progress snapshots, degradation states, and proof refs. |
| [`surface_projection.json`](./surface_projection.json) | Left rail, header, inspector, digest, command refs, preview labels, rate limits, exact reopen refs, and profile controls. |
| [`support_export.json`](./support_export.json) | Metadata-only support export for package versions, profile state, and progress state with raw bodies omitted. |
| [`mutation_preview_path.yaml`](./mutation_preview_path.yaml) | Import-profile exercise proof that teaching mutation uses the command registry, preview sheet, approval path, rollback semantics, and evidence ref. |
| [`degraded_sources.yaml`](./degraded_sources.yaml) | Cached-docs and graph-unavailable degradation proof with inactive placeholders and citations. |
| [`progress_user_owned.yaml`](./progress_user_owned.yaml) | User-owned progress proof for local-only default, optional sync, hidden-read denial, inspectability, and redacted export. |

## Regenerate

```sh
cargo run -q -p aureline-shell --bin aureline_shell_learning_mode_beta -- manifest \
  > fixtures/help/m3/guided_tours/manifest.json
cargo run -q -p aureline-shell --bin aureline_shell_learning_mode_beta -- surfaces \
  > fixtures/help/m3/guided_tours/surface_projection.json
cargo run -q -p aureline-shell --bin aureline_shell_learning_mode_beta -- support-export \
  > fixtures/help/m3/guided_tours/support_export.json
```

## Verification

```sh
cargo test -p aureline-shell --test learning_mode_beta_fixtures
cargo run -q -p aureline-shell --bin aureline_shell_learning_mode_beta -- validate
```
