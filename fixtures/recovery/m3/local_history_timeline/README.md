# Local-history timeline alpha fixtures

This corpus protects the local-history compare, restore, and export
timeline projection. Each row carries the same fidelity label on the row
and its visible actions so timeline UI and support export can quote one
vocabulary.

The four protected labels are `exact`, `compatible`, `layout_only`, and
`evidence_only`. Evidence-only rows must keep restore disabled and must
not claim that a live session or privileged run resumed.
