# Navigation Target And Hierarchy Contract Fixtures

This corpus backs the stable metadata contract for navigation targets, reference occurrences, hierarchy edges, disambiguation sets, and rename previews.

The fixture keeps source bodies out of the packet and verifies that relation kinds, access kinds, provider/source classes, freshness, confidence, ambiguity, fallback modes, hierarchy edge classes, and rename blocked/omitted candidate truth survive across editor, search, graph, review, AI, support export, and CLI/headless projections.
