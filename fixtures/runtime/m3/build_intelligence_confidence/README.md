These fixtures drive the runtime build-intelligence beta tests. They cover:

- native adapter, structured protocol, build-event stream, structured output import, and heuristic fallback lanes;
- adapter health reasons with precise degraded or partial causes;
- imported/replayed receipts that remain separate from current live workspace inspection;
- discovery refresh diffs split into added, removed, renamed, downgraded-confidence, newly-heuristic, newly-exact, and now-unresolved buckets.
