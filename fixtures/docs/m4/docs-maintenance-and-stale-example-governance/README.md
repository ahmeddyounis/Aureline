# Docs Maintenance And Stale Example Governance Fixtures

These fixtures exercise the stable docs-maintenance governance packet.

- `baseline_stable.json` validates the complete stable packet.
- `rendered_preview_claims_canonical_blocks_stable.json` proves rendered preview cannot become canonical source.
- `suppression_loses_actor_blocks_review.json` proves stale-example suppressions require actor, reason, expiry, and evidence refs.
- `projection_drops_publish_boundary_blocks_stable.json` proves consumer projections must preserve publish-boundary and browser-handoff state.
