# OS appearance transition cases

Worked cases for OS-triggered appearance changes (theme, contrast/forced colors,
accent, text scale, reduced motion) across claimed desktop profiles.

These fixtures are referenced by:

- `docs/design/os_appearance_change_matrix.md`
- `artifacts/design/appearance_live_change_matrix.yaml`

They are design-side “truth cases” used by QA and support exporters. They are
not runtime integration tests.
