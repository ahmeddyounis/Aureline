# Palette mapping examples

This directory contains **machine-readable palette mapping examples** for the
semantic token domains defined in:

- `docs/design/semantic_token_domains_and_palette_contract.md`
- `artifacts/design/semantic_token_domains.yaml`

Each JSON file is a `palette_mapping_example_record` validated by:

- `schemas/design/palette_mapping_row.schema.json`

The examples are intentionally small and focus on *meaning ownership* and
*override rules* rather than shipping a complete theme pack.
