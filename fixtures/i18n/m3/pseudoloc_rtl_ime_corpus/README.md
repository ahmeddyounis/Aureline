# Dense I18n Conformance Corpus

This fixture set is the beta dense-surface corpus for pseudolocalization, RTL,
bidi, IME composition, CJK/font fallback, locale fallback, translated-surface
parity, and text expansion.

Files:

- `manifest.json` is the machine-readable corpus generated by
  `aureline-i18n`.
- `review_export.json` is the exportable review packet derived from the
  manifest.

Regenerate and validate:

```sh
cargo run -q -p aureline-i18n --bin aureline_i18n_locale_pack_beta -- dense-corpus > fixtures/i18n/m3/pseudoloc_rtl_ime_corpus/manifest.json
cargo run -q -p aureline-i18n --bin aureline_i18n_locale_pack_beta -- dense-review > fixtures/i18n/m3/pseudoloc_rtl_ime_corpus/review_export.json
cargo test -p aureline-i18n --test pseudoloc_rtl_ime_corpus --locked
```

The corpus intentionally covers dense product workflows rather than simple
forms: editor assist, palette, settings, trees, tables, logs, terminal, review,
guided tours, and docs/help.
