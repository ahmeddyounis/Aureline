# Workflow bundle lifecycle visibility fixture cases

This directory contains workflow-bundle lifecycle visibility cases that
exercise successor guidance, certification expiry visibility, mirror/offline
posture visibility, removed artifact family handling, support-class narrowing,
and local override/drift interactions.

The normative narrative lives at:

- `docs/workflow/bundle_lifecycle_visibility_contract.md`

These cases are intentionally cross-cutting: they include the minimum
fragments needed to reason about what a Start Center card/detail surface,
an update/export/support surface, and a certification sheet must preserve.
