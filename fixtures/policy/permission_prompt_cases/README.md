# Permission prompt cases

Worked examples for the permission prompt lifecycle and reapproval
trigger packet defined by:

- `docs/policy/permission_prompt_state_machine.md`
- `schemas/policy/permission_prompt_event.schema.json`

Each case is a reviewer-facing narrative plus a small event sequence.
The events are export-safe and avoid raw prompt bodies, raw policy
payloads, raw URLs, raw paths, and secret material.
