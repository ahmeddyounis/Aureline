# Toast cases

Worked fixtures for the toast acknowledgement, undo, and durable-linkback
contract in [`/docs/ux/toast_contract.md`](../../../docs/ux/toast_contract.md)
and [`/schemas/ux/toast_event.schema.json`](../../../schemas/ux/toast_event.schema.json).

These fixtures are intentionally small and focused: they exist so
reviewers can distinguish acknowledgement-only toasts from durable
surfaces mechanically, and so that undo and durable rediscovery posture
can be audited without reading UI screenshots.
