# Command palette diagnostics fixtures

Reviewable fixtures for the beta command-palette diagnostics
projection implemented in
[`crates/aureline-shell/src/palette/diagnostics_beta.rs`](../../../../crates/aureline-shell/src/palette/diagnostics_beta.rs).

The JSON files are generated by the headless inspector
[`crates/aureline-shell/src/bin/aureline_shell_command_palette_diagnostics.rs`](../../../../crates/aureline-shell/src/bin/aureline_shell_command_palette_diagnostics.rs).
The inspector is the mint-from-truth path for the checked-in JSON, so
palette rows, deep-link reviews, support export, and parity examples
stay aligned.

## Index

| Fixture | Coverage |
| --- | --- |
| [`page.json`](./page.json) | Full diagnostics pack with row anatomy, action footer, preview pane, diagnostics-sheet, invocation-preview, privacy, warm-open, and no-bypass summaries. |
| [`support_export.json`](./support_export.json) | Metadata-safe support export that quotes command IDs and reason codes while omitting raw query text and private values. |
| [`../../../../artifacts/commands/m3/palette_parity_examples.json`](../../../../artifacts/commands/m3/palette_parity_examples.json) | Parity examples derived from the same pack for palette, menu, keybinding, onboarding hint, CLI, AI, and support surfaces. |

## Fixture Rules

- Every row is a command row with a canonical `cmd:` ID, origin badge,
  shortcut or `Unassigned`, side-effect class, automation labels, and
  automation display cues.
- Disabled rows carry a reason code and expose a diagnostics sheet.
- Preview-required rows route to the invocation preview sheet and
  preserve approval state before apply.
- Footer actions cover primary run or preview, split or alternate
  open, copy command ID, copy CLI/headless form, add to recipe, and
  why-not automatable.
- Deep-link reviews preserve strict no-bypass guards for trust,
  policy, permission, preview, approval, credential broker, execution
  context, freshness, capability class, and result schema checks.
- The support export omits raw query text, raw argument values, and
  private workspace paths.

## Regenerate

```sh
cargo run -q -p aureline-shell --bin aureline_shell_command_palette_diagnostics -- pack \
  > fixtures/ux/m3/command_palette_diagnostics/page.json
cargo run -q -p aureline-shell --bin aureline_shell_command_palette_diagnostics -- support-export \
  > fixtures/ux/m3/command_palette_diagnostics/support_export.json
cargo run -q -p aureline-shell --bin aureline_shell_command_palette_diagnostics -- parity-examples \
  > artifacts/commands/m3/palette_parity_examples.json
```

## Verification

```sh
cargo test -p aureline-shell --test command_palette_diagnostics_beta_fixtures
cargo run -q -p aureline-shell --bin aureline_shell_command_palette_diagnostics -- validate
```
