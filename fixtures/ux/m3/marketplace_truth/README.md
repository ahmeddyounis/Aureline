# Marketplace Truth Fixture Corpus

Reviewable fixtures for the marketplace truth projection implemented in
[`crates/aureline-shell/src/extensions/marketplace/`](../../../../crates/aureline-shell/src/extensions/marketplace/).

The fixtures are generated by the headless inspector
[`aureline_shell_marketplace_truth`](../../../../crates/aureline-shell/src/bin/aureline_shell_marketplace_truth.rs).
The inspector reads the checked-in catalog descriptor and the generated
compatibility report, then emits the same rows consumed by shell, CLI, and
support-export surfaces.

## Index

| Fixture | Coverage |
| --- | --- |
| [`rows.json`](./rows.json) | Marketplace rows with controlled lifecycle badges, generated-report compatibility labels, support-class chips, trust chips, mirrorability, and install-review refs. |
| [`support_rows.json`](./support_rows.json) | Metadata-safe support rows quoting each marketplace row. |
| [`page.json`](./page.json) | Full shell page wrapper with summary counts and the complete controlled badge vocabulary. |

## Regenerate

```sh
cargo run -q -p aureline-shell --bin aureline_shell_marketplace_truth -- page         > fixtures/ux/m3/marketplace_truth/page.json
cargo run -q -p aureline-shell --bin aureline_shell_marketplace_truth -- rows         > fixtures/ux/m3/marketplace_truth/rows.json
cargo run -q -p aureline-shell --bin aureline_shell_marketplace_truth -- support-rows > fixtures/ux/m3/marketplace_truth/support_rows.json
```

## Verification

```sh
cargo test -p aureline-shell --test marketplace_truth_beta_fixtures
cargo run -q -p aureline-shell --bin aureline_shell_marketplace_truth -- validate
```
