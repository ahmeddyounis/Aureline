# Extension inspector fixtures

These fixtures are generated by the shell headless inspector:

```text
cargo run -q -p aureline-shell --bin aureline_shell_extension_inspectors -- page
cargo run -q -p aureline-shell --bin aureline_shell_extension_inspectors -- support-export
```

- `page.json` contains the shell page with permission and settings inspectors.
- `support_export.json` contains the metadata-safe support export projected from the same rows.

The Rust fixture test asserts both files match the seeded builder exactly.
