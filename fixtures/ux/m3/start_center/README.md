# Start Center and workspace switcher beta fixtures

Reviewable fixtures for the beta Start Center and workspace-switcher
projection in
[`crates/aureline-shell/src/start_center/beta.rs`](../../../../crates/aureline-shell/src/start_center/beta.rs).

The JSON files are generated by the headless inspector
[`crates/aureline-shell/src/bin/aureline_shell_start_center.rs`](../../../../crates/aureline-shell/src/bin/aureline_shell_start_center.rs).
The inspector is the mint-from-truth path for the checked-in JSON, so
live rows, CLI output, and support-export rows stay aligned.

## Index

| Fixture | Coverage |
| --- | --- |
| [`primary_actions.json`](./primary_actions.json) | The five first-run actions in required order, all keyboard reachable and renderable before sign-in or network readiness. |
| [`rows.json`](./rows.json) | Start Center pinned/recent rows across reachable, missing-path, moved-root, SSH-unavailable, devcontainer-unavailable, and managed-reauth states. |
| [`switcher_rows.json`](./switcher_rows.json) | Workspace-switcher rows projected from the same recent-work entries, with cancel and reopen-prior return paths. |
| [`privacy_modes.json`](./privacy_modes.json) | Default, hide-paths, hide-recent-work, and open/clone-only privacy projections. |
| [`page.json`](./page.json) | Full page projection with primary actions, privacy records, open-window row, Start Center rows, switcher rows, support rows, and summary. |
| [`support_export.json`](./support_export.json) | Support-export wrapper that mirrors page rows and excludes raw secret material. |

## Regenerate

```sh
cargo run -q -p aureline-shell --bin aureline_shell_start_center -- primary-actions > fixtures/ux/m3/start_center/primary_actions.json
cargo run -q -p aureline-shell --bin aureline_shell_start_center -- rows            > fixtures/ux/m3/start_center/rows.json
cargo run -q -p aureline-shell --bin aureline_shell_start_center -- switcher-rows   > fixtures/ux/m3/start_center/switcher_rows.json
cargo run -q -p aureline-shell --bin aureline_shell_start_center -- privacy-modes   > fixtures/ux/m3/start_center/privacy_modes.json
cargo run -q -p aureline-shell --bin aureline_shell_start_center -- page            > fixtures/ux/m3/start_center/page.json
cargo run -q -p aureline-shell --bin aureline_shell_start_center -- support-export  > fixtures/ux/m3/start_center/support_export.json
```

## Verification

```sh
cargo test -p aureline-shell --test start_center_switcher_beta_fixtures
cargo run -q -p aureline-shell --bin aureline_shell_start_center -- validate
```
