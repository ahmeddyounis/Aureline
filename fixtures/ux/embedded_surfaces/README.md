# Embedded surface fixtures

This directory holds small, shell-consumable fixtures for embedded-surface
render contracts.

Unlike the broader boundary-case corpus under `fixtures/ux/embedded_boundary_cases/`,
these files are intended to be directly deserialized by shell/runtime code for
baseline development and reviewer walkthroughs.

- `docs_help_boundary_contract_card.json` — docs/help embedded boundary card
  fixture used by the desktop shell baseline.
