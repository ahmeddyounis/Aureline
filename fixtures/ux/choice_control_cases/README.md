# Choice control cases

Worked fixtures for
[`docs/ux/toggle_and_choice_control_contract.md`](../../../docs/ux/toggle_and_choice_control_contract.md)
and
[`schemas/ux/choice_control_state.schema.json`](../../../schemas/ux/choice_control_state.schema.json).

The corpus covers:

- `live_switch_immediate_apply.json` — a reversible switch that applies
  immediately.
- `deferred_apply_checklist.json` — a staged checklist whose changes do
  not apply until `Apply`.
- `connection_mode_radio_group.json` — mutually exclusive connection
  modes with mixed-selection disclosure.
- `view_mode_segmented_control.json` — compact segmented view mode
  switch with roving-focus keyboard behavior.
- `policy_locked_choice.json` — a policy-locked selection that remains
  truthfully readable and explainable.

Fixtures carry opaque refs and review-safe copy only.
