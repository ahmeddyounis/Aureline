# Native file affordance cases

Worked fixtures for the contract frozen in:

- `docs/ux/native_file_affordance_contract.md`
- `schemas/platform/native_file_affordance.schema.json`

These cases exist to keep native open/save dialogs, reveal-in-shell, file
associations, open-from-terminal, drag/drop, clipboard, and browser-originated
open requests from drifting into per-OS behavior that bypasses identity binding,
trust/policy review, save-token write posture, or safe-preview/suspicious-content
disclosure.

Each fixture is a `native_file_affordance_case_record`.
