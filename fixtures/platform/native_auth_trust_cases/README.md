# Native auth/trust conformance fixtures

This directory contains worked YAML fixtures that exercise the native browser
handoff/return, proxy/PAC inheritance, trust-store layering, and credential-store
degraded-state rules for claimed desktop profiles.

The fixtures are reviewer drill cases referenced by:
