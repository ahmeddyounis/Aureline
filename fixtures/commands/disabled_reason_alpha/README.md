# Alpha Disabled-Reason Fixtures

These fixtures publish the alpha disabled-reason vocabulary proof for command
discoverability surfaces. They map the user-facing cause families required by
the alpha wedge to the canonical `disabled_reason_code` values already used by
the command enablement engine.

`manifest.json` is consumed by the `aureline-commands` alpha registry tests.
It proves focus, selection, lifecycle state, missing dependency, policy,
entitlement, and remote/host mismatch cases remain typed, repairable, and
export-safe.
