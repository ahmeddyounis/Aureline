# Archetype Detection Fixtures

Compact fixture set for beta workspace archetype detection and first-useful-work
routing. Full schema-shaped route records remain in
[`fixtures/workspace/entry_routes/`](../../entry_routes/); these files keep the
detector-facing matrix easy to review.

Each case names the expected detection outcome, evidence freshness posture,
readiness bucket counts, route surface, and bypasses that must stay available.
