# Install Profile Cards And Repair Fixtures

These fixtures exercise the beta install-profile card, side-by-side import,
rollout-ring, repair, verify, and uninstall contracts.

- `profile_cards_packet.json` carries the Windows Stable, Preview, portable,
  and managed profile cards, the Stable-to-Preview import sheet, and Canary,
  Pilot, Broad, and LTS rollout rows.
- `repair_verify_uninstall_packet.json` carries managed/silent repair, verify,
  verify-failure, and uninstall diagnostics with copyable install IDs,
  timestamps, state-root refs, support refs, and failure summaries.
- `*_support_export.json` files are generated by the Rust packet projections
  and remain metadata-only.

Run:

```bash
cargo test -p aureline-install --test profile_cards_and_repair_beta
```
