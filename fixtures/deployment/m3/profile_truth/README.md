# Deployment-profile continuity corpus: profile-truth cases

This directory holds the per-row deployment-profile pages that exercise
the matrix of marketed beta deployment rows (individual-local,
self-hosted / sovereign, enterprise-online hybrid, enterprise-online
mirror-only, air-gapped, managed-cloud) across the desktop, CLI/headless,
companion-handoff, and support-export surface lenses.

Each `cases/<case_id>.json` file is a
[`DeploymentProfilePage`](../../../../crates/aureline-shell/src/deployment_profile/mod.rs)
record. Every page MUST pass `DeploymentProfilePage::audit()` with an
empty defect set; the corpus replay test
(`crates/aureline-shell/tests/deployment_profile_corpus_fixtures.rs`)
enforces that.

`packet.json` is the composite corpus packet generated by
`crates/aureline-shell/src/deployment_profile/corpus.rs`. It binds the
case set to the outage-drill set under
`fixtures/deployment/m3/control_plane_vs_data_plane/drills/`, the
residual-dependency matrix, and the coverage summary.

To regenerate the fixtures after editing the builder:

```bash
cargo run -q -p aureline-shell --bin aureline_shell_deployment_profile_corpus -- packet \
  > fixtures/deployment/m3/profile_truth/packet.json
for id in $(cargo run -q -p aureline-shell --bin aureline_shell_deployment_profile_corpus -- cases \
  | jq -r '.[].case_id'); do
  cargo run -q -p aureline-shell --bin aureline_shell_deployment_profile_corpus -- case "$id" \
    > "fixtures/deployment/m3/profile_truth/cases/$id.json"
done
```

Reviewers consume the rendered release-evidence excerpt at
[`artifacts/release/m3/deployment_profile_conformance_report.md`](../../../../artifacts/release/m3/deployment_profile_conformance_report.md)
and the per-profile residual-dependency matrix JSON at
[`artifacts/release/m3/residual_dependency_matrix.json`](../../../../artifacts/release/m3/residual_dependency_matrix.json).

Beta-exit qualification rules live in
[`docs/ops/m3/deployment_profile_claim_qualification.md`](../../../../docs/ops/m3/deployment_profile_claim_qualification.md).
