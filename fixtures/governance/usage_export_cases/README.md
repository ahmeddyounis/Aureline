# Usage export packet fixtures

This directory contains worked cases for the packet-level usage export
and offboarding contract.

- Contract: `docs/governance/usage_export_and_offboarding_contract.md`
- Packet schema: `schemas/governance/usage_export_record.schema.json`
- Row schema: `schemas/managed/usage_export_row.schema.json`
- Exit packet schema: `schemas/governance/offboarding_exit_packet.schema.json`

The fixtures are YAML for readability; each file pins its JSON Schema
via a `yaml-language-server` `$schema` directive.
