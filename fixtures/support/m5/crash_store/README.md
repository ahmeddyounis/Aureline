# M5 crash-store fixtures

These fixtures exercise the local crash-store viewer packet used by
`aureline-support`.

| Fixture | Purpose |
| --- | --- |
| `packet.json` | Canonical local-first crash-store viewer packet covering the claimed M5 host families. |
| `expired_dump_metadata_only.json` | Variant where the query-runtime raw dump expired but metadata review remains available. |
| `manifest.yaml` | Protected fixture manifest and assertions. |
