# Placeholder and late-copy worked cases

This fixture corpus exercises the translation-safe content-ops contract
in [`docs/copy/translation_safe_content_ops_contract.md`](../../../docs/copy/translation_safe_content_ops_contract.md).

Fixtures:

- `translator_note_preservation.yaml` - placeholder-rich docs/help copy
  where required translator notes are preserved.
- `screenshot_caption_metadata.yaml` - screenshot caption metadata bound
  to source product messages and a capture build.
- `truncation_review_failure.yaml` - launch-critical recovery copy that
  fails truncation review and is denied until repaired.
- `controlled_late_copy_change.yaml` - controlled after-freeze copy
  change with owner, reviewers, verification notes, delta scope, and
  downstream surfaces.
- `source_language_fallback.yaml` - trust/recovery copy falling back to
  source language with disclosure and escape hatches.
