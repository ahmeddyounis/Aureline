# Community Handoff Boundary

Aureline community links must disclose the destination boundary before the user
leaves the product.

| Lane | Destination class | Boundary |
|---|---|---|
| Public issue | `public` | Open-project issue visible publicly after preview |
| Security disclosure | `official_authenticated` | Private security lane only |
| Docs feedback | `public` | Exact docs page and anchor preserved |
| RFC discussion | `community` | Feedback only, not a commitment |
| Community support | `community` | Best-effort community help |
| Vendor/private support | `vendor_managed` | Provider-owned support boundary |

Blocked browser launches and offline states keep drafted text, selected
attachments, redaction settings, and target class durable. The available actions
are retry browser, export local packet, or open later.
