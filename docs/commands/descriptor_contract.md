# Descriptor and registry contract (entrypoint)

The canonical contract for command descriptors and invocation-session packets
is frozen in `docs/commands/command_descriptor_contract.md`.

The machine-readable boundary is `schemas/commands/command_descriptor.schema.json`,
and the seeded canonical registry is `artifacts/commands/command_registry_seed.yaml`.

The runtime owner of the canonical registry types is `crates/aureline-commands`.
