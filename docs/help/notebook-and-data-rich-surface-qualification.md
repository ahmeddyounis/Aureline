# Notebook and Data-Rich Surface Status

Notebook surfaces marked Stable show document trust, kernel/runtime trust, and
output trust separately. A notebook can remain open, editable, searchable, and
reviewable with no selected kernel; prior outputs stay visible as stale,
captured, imported, or safe-preview evidence when live execution is unavailable.

Data tables, result grids, chart summaries, experiment handoffs, and
API/database response viewers are Preview in this build. They support safe
inspection and redacted export posture, but they do not imply Stable database,
API, charting, or experiment-platform depth.
