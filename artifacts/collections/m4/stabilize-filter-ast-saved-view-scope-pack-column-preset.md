# M4 Stable Dense-Collection Contract Artifact

The stable packet certifies one governed collection contract across dense
surfaces:

- search result export with hidden policy rows and explicit identity selection
- provider-backed review/admin batch approval using prior query snapshot truth
- package/test/diagnostics grid with stable selected identities
- notebook data grid with streaming, partial results, and all-matching export

The packet preserves filter AST, saved-view, query-history, scope-pack,
column-preset, scope-counter, selection, and batch-review objects. Consumer
projections cover desktop UI, CLI/headless output, export packets, support
capture, and accessibility tree narration.

Negative fixtures prove stable publication blocks when:

- notebook/data-grid coverage is missing
- provider-owned protected actions omit a batch-review sheet
- scope vocabulary drops required count terms such as `approx.`, `hidden by
  policy`, or `outside current filter`
