# Graph topology and explainer packets

This directory carries reviewable graph-understanding export packets for topology maps, impact explorers, and cited codebase explainers.

The packets preserve:

- graph node and edge ids across canvas, list, table, and export paths
- the shared scope vocabulary: `Current repo`, `Selected workset`, `Full workspace`, `Remote cache`, `Outside current scope`
- controlled impact reasons: `exact_edge`, `shared_target`, `ownership_rule`, `generated_linkage`, `heuristic_similarity`, `policy_coupling`
- cited explainer claims, generated-vs-curated labels, omission disclosures, and freshness/provenance strips
