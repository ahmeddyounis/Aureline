# Proof packets (M1)

This directory holds reviewer-facing proof packets for M1 proof lanes.

Rules:

- Each proof lane listed in `artifacts/milestones/m1/artifact_index.yaml` MUST
  resolve to exactly one owning packet in this directory.
- A packet can link to canonical sources elsewhere in the repo, but the lane's
  review-facing evidence links should remain stable and discoverable via the
  artifact index.
